vmp

取这名字真的吓我一跳，我以为加了vmp壳，结果查壳一看，标志位也没有，好像没加壳，打开一看就像普通vm题，纯被吓了

opcode的映射表是这样的

Opcode 语义 编码格式 说明
0x3C mov reg, imm32 3c reg imm32_le 给寄存器赋立即数
0xA3 in reg a3 reg 从输入流取 1 字节到寄存器
0x19 load dst, [base+off] 19 dst base off 从 VM 内存读 1 字节到寄存器
0x1A store [base+off], src 1a src base off 把寄存器低字节写到 VM 内存
0x4E xor dst, src 4e dst src dst ^= src
0x5E mov dst, src 5e dst src 寄存器间赋值
0x6B cmp a, b 6b a b 比较两个寄存器，设置标志位
0xE3 je imm32 e3 imm32_le 若相等标志为 1，则跳转到字节码偏移imm32
0xA1 out reg a1 reg 输出寄存器低字节对应字符
0xFE set r15, imm8 fe imm8 给r15赋值，样本里当作状态码
0xFF halt ff 停止执行 VM

直接分析这vmp段，很明显是一段opcode

最主要理解0x3C和0x4E

0x3C可以理解为把立即数赋值给寄存器

0x4E就是异或
